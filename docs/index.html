[ Live Proxy ]

Stream konten secara langsung melalui server kami. Endpoint ini ideal untuk akses cepat tanpa perlu menyimpan file di server.

GET /proxy

Query Parameters:

  • url (required): URL lengkap dari sumber daya yang ingin Anda proxy.

Contoh Penggunaan (cURL):

curl "/proxy?url=https://api.ipify.org?format=json"

[ Smart Downloader ]

Untuk file besar atau unduhan yang memerlukan header khusus. Sistem ini bekerja di latar belakang dan secara otomatis mengarahkan Anda ke file ketika sudah siap.

// Langkah 1: Meminta Download

POST /api/request-download

Body (JSON):

{
    "link": "https_URL_FILE_TARGET",
    "filename": "nama_file_yang_diinginkan.mp3",
    "headers": { "Referer": "https://situs-asal.com/" }
}

Contoh Response:

Server akan langsung merespons dengan URL unduhan pintar (`downloadUrl`).

{
    "success": true,
    "message": "Permintaan diterima! Buka `downloadUrl` untuk memulai unduhan.",
    "jobId": "a1b2c3d4-...",
    "downloadUrl": "/dl/a1b2c3d4-..."
}

// Langkah 2: Buka Smart Redirect URL

Cukup buka downloadUrl yang Anda terima di browser. Halaman akan menampilkan status proses dan me-refresh secara otomatis. Ketika unduhan selesai, Anda akan langsung diarahkan ke file tersebut.

GET /dl/:jobId
