[ Live Proxy ]

Stream konten secara langsung melalui server kami. Gunakan endpoint ini untuk akses sinkron dan cepat tanpa penyimpanan file.

GET /proxy

Query Parameters:

  • url (required): URL lengkap dari sumber daya yang ingin Anda proxy.

Contoh Penggunaan (cURL):

curl "/proxy?url=https://api.ipify.org?format=json"

[ Async Downloader ]

Untuk file besar atau download yang butuh waktu. Sistem ini bekerja di latar belakang. Anda meminta download, lalu memeriksa statusnya secara berkala.

// Langkah 1: Meminta Download

POST /api/request-download

Body (JSON):

{
    "link": "https_URL_FILE_TARGET",
    "filenameHint": "nama_file_yang_diinginkan.zip",
    "headers": { "Optional-Header": "value" }
}

Contoh Response:

{
    "success": true,
    "message": "Permintaan diterima! Gunakan pollingUrl untuk cek status.",
    "jobId": "...",
    "pollingUrl": "/api/status/..."
}

// Langkah 2: Cek Status (Polling)

GET /api/status/:jobId

Gunakan pollingUrl dari response sebelumnya untuk memeriksa status job.

Contoh Response (Completed):

{
    "success": true,
    "status": "completed",
    "message": "Download Selesai!",
    "downloadLink": "/downloads/...",
    "filename": "..."
}
